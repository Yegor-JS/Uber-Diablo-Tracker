This is a Diablo Clone tracker to keep us all posted about Terror, that supplies Sanctuary with Annihiluses.

This tracker relies on a public API, that, in turn, relies on actual people's reports. And some people are trolls, keep that in mind.

If you allow this website to show you notifications, you'll be notified about any progress changes that happened anywhere in Sanctuary regarding Diablo Clone. This tracker checks for updates every 30 seconds. The table gets refreshed automatically.

Pro tip: the history of changes we managed to capture is kept in the browser console (F12) for as long as you don't close the tab.
